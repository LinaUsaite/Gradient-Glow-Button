# Gradient Glow Button

A custom CSS button with a gradient glow effect created using pseudo-elements and hover interactions.

## Features
- Centered layout using Flexbox
- Gradient border created with ::after pseudo-element
- Smooth glow effect on hover
- Rounded button design

## Technologies
- HTML
- CSS (Flexbox, pseudo-elements, gradients)

## How it works
The button uses a pseudo-element positioned behind it to create a gradient border effect. On hover, a glowing box-shadow is applied to enhance the visual interaction.

## Purpose
This project was created to practice CSS layout techniques, pseudo-elements, and visual UI effects.
